## Description

Nestjs Fastify Manager Scaffolding build using Nestjs, Fastify, and RabbitMQ

## Installation

```bash
$ npm install

# Configure env by copying env.[environtment] to .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
